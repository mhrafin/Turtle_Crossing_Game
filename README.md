## Run Locally

Clone the project

```bash
  git clone https://github.com/mhrafin/Turtle_Crossing_Game.git
```

Go to the project directory

```bash
  cd Turtle_Crossing_Game
```

Run the game (main.py)

```bash
  python3 main.py
```
